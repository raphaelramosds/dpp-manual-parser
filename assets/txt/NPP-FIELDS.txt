FIELD "NOME_POCO_ANP" IS TEXTO(20) REQUIRED 
RULES ARE O "Nome do Poço ANP” informado deve estar cadastrado nas bases de dados da
ANP. Um exemplo de preenchimento do nome do poço seria "9-RO-234-TESTE”.

FIELD "NUMERO_FASE" IS INTEIRO(2) REQUIRED 
RULES ARE O campo "Número Fase” de cada Cimentação só será considerado válido se o
número tiver sido declarado na lista de Fases dessa carga.

FIELD "CODIGO_SONDA" IS INTEIRO(10) REQUIRED 
RULES ARE O campo "Sonda" só será considerado válido se estiver cadastrado na base de
dados do SIGEP com o status "Aprovado”. Além de estar cadastrado no SIGEP o tipo de sonda
informada também deve ser compatível com o campo "Poço Submarino”. Caso o campo "Poço
Submarino” tenha sido preenchido com o valor "S” os tipos de sondas válidos são: "Fixa", "Auto-
elevável", "Semi-submersível”, "Navio-sonda” (ambiente Marítimo). Caso o campo "Poço
Submarino” tenha sido preenchido com o valor "N" os tipos de sondas válidos são: "Rotativa”,
"Roto-Pneumática”, "Completação/Workver” (ambiente terrestre). Essas informações são
apresentadas no menu SIGEP Exploração > CADASTROS > Sondas.

FIELD "DATA_PREVISAO_INICIO" IS DATA(10) REQUIRED 
RULES ARE Preenchimento no formato: DD/MM/AAAA. O campo "Previsão Infcio” deve ser
maior que a data informada no campo "Previsão Fim” relativa a última fase informada para 0 poço.

FIELD "DATA_PREVISAO_FIM" IS DATA(10) REQUIRED 
RULES ARE Preenchimento no formato: DD/MM/AAAA. O campo "Previsão Fim" só será válido
se for maior ou igual ao campo "Previsão Início”. A comparação deverá ser feita entre os campos
"Previsão Fim” e "Previsão Início” de um mesmo registro da lista de registros "Outras Atividades”.
